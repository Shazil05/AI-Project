
# Description #
This repo comprises an AI system project that transforms reading passages into interactive quizzes by generating questions, correct answers, distractors, and hints using the `RACE` dataset. It relies on traditional ML models like SVM and Logistic Regression, which are vectorized via One-Hot Encoding, and is presented through a multi-screen Streamlit application.

## Dataset ##
[https://www.kaggle.com/datasets/ankitdhiman7/race-dataset]
